실제 매체 연동 설정

메타 · 유튜브 · 틱톡 API를 연결하면 저효율 캠페인이 진짜로 일시정지됩니다.

API 서버 확인 중…

1단계 · 배포 (팀 내부용)

# Railway 등에 배포 후 팀 URL 공유
# 또는 로컬: ./run_api.sh

웹+API가 한 서버에서 실행됩니다. 팀원은 URL + 비밀번호만 알면 됩니다.

2단계 · .env 설정

cp .env.example .env
# .env 파일 편집 후:
API_MODE=live

메타 (Facebook / Instagram)

  1. Graph API Explorer에서 토큰 발급
  2. 권한: ads_management, ads_read
  3. 광고계정 ID를 META_AD_ACCOUNT_IDS에 입력
테스트 대기

유튜브 (Google Ads)

  1. Google Cloud OAuth 클라이언트 생성
  2. python3 scripts/google_ads_auth.py 실행 → refresh token 발급
  3. Developer Token + Customer ID 설정
테스트 대기

틱톡

  1. TikTok API Portal 앱 생성
  2. Access Token 발급
  3. TIKTOK_ADVERTISER_IDS에 광고주 ID 입력
테스트 대기
대시보드로 이동
⚠️ 실제 중단 주의

API_MODE=live 상태에서 자동 중단을 켜면, 조건에 맞는 캠페인에 PAUSED / DISABLE 요청이 실제로 전송됩니다. 먼저 연결 테스트로 계정·캠페인 수를 확인하세요.
